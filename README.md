# 🌐 Google Login OAuth with Spring Security, React & MySQL

![Poster](https://media2.dev.to/dynamic/image/width=1280,height=720,fit=cover,gravity=auto,format=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Flnvp60bmcmubd0xauq7j.png)

A full-stack web application that implements Google OAuth 2.0 login using **Spring Boot** and **Spring Security** for backend authentication, **React** for the frontend interface, and **MySQL** for persistent user data storage.

---

## ✨ Features

- 🔐 Secure Google OAuth2 Login
- 💻 React Frontend with Protected Routes
- 🛡 Spring Boot Backend with Spring Security
- 💾 MySQL Database Integration
- ✅ JWT Token Management (optional extension)

---

## 🛠️ Technologies Used

- **Frontend:** React, Axios, Bootstrap
- **Backend:** Spring Boot, Spring Security, OAuth2 Client
- **Database:** MySQL
- **Other:** Maven, Postman, JWT (optional)
